hash table
Linear List(Singlylinked, doublylink, array list, vector), 1-D list - does not support search well. Searching time: O(N)
we need dynamic storage with quick search!
hashtable is such a thing! Support: 1) grow 2) shrink 3) iteration/traversal 4) search

e.g. H: car plate -> [0, 20] H (car) = car plate number %20
function values can be used as the offset in the a storage (for positioning)
