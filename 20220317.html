Merge Sort - advanced sorting algorithm - uses the divide-and-conquer strategy (like the quicksort) Steps to sort: 1) partition perfectly 2) recursively merge the subsets 3) perform merge
Ideal divide-and-conquer is to have a balanced partition - drawback of quicksort is that we can't guarantee the partition is balanced
We want a perfect balanced partition at all cost. T(n) = 2T(n/2) + O(n) --> T = n log(n)
we scarify the nice property that any element in 1st part <= 2 any in 2nd part, to exchange a perfect balanced partition
